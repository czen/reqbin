Quick and dirty implementation of a request emulator
Specify redirect as protocol-address. http-127.0.0.1:8080 will redirect to http://127.0.0.1:8080
  • /redirect/:location -- redirect to given address -- /redirect/http-127.0.0.1
  • /redirect/local -- redirects to http://127.0.0.1
  • GET /redirect307/:location -- 307 redirects to given address
  • POST /redirect307/:location -- 307 redirects to given address
